jGauge 0.3.0 Alpha 3 - Live Demonstration

jGauge is an open-source JavaScript dial gauge designed to be simple and powerful!

Please note this is an alpha release. Version 0.3.0 is still under development use with caution! ;-) If you find jGauge useful a link back to my blog is appreciated.

See the main project page for more information and downloads.

See the documentation for developer references and usage examples.

Set the value:

0 2.5 5 8 10

Bump the value:

+ 0.1 + 1 + 100 + 1000 + 106 + 109 + 1012
- 0.1 - 1 - 100 - 1000 - 106 - 109 - 1012

Set the number of ticks:

0 (None) 2 5 6 11
